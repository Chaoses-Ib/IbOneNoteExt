# IbOnenoteExt
Language: [English](README.md), [简体中文](README.zh-Hans)

## 功能
* 修改 Calibri 字体为微软雅黑。
	* 影响输入和样式字体。
	* 影响现存 Calibri 文本的显示。
	* 导致不能再向文本应用 Calibri。

## 支持
* Microsoft OneNote 2016 (16.0.5014.1000) MSO (16.0.5095.1000) 64-bit

## 用法
1. 关闭 OneNote，等几分钟。
1. 把 version.dll 放进 C:\Program Files\Microsoft Office\Office16。
1. 重新启动 OneNote。
